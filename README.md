🚀 Football Analytics Platform

Interactive football analytics platform built around Liverpool FC’s 2024/25 Premier League-winning season, designed to explore performance through data-driven insights, tactical analysis, and interactive visualisations.

⸻

📌 Project Overview

This project was built to transform raw football data into an interactive exploration tool that reveals patterns in team performance, player contributions, and tactical structure.

Rather than presenting static dashboards, the focus is on enabling users to explore the season from multiple analytical perspectives and understand the story behind the numbers.

⸻

🎯 Key Features

📊 Season performance trends and key metrics
⚽ Expected Goals (xG) and Expected Assists (xA) analysis
🎯 Attacking patterns and chance creation insights
🛡️ Defensive performance breakdowns
🧠 Tactical formations and team structure visualisations
🔄 Player and team comparisons
📈 Interactive dashboards focused on exploration and storytelling

⸻

🛠️ Tech Stack

* React
* JavaScript
* Tailwind CSS
* Recharts

📊 Key Insights

* Identified key attacking patterns driving goal creation across the season
* Analysed expected vs actual performance in critical match phases
* Evaluated defensive stability across different tactical setups
* Compared player contributions using advanced metrics like xG and xA

⸻
